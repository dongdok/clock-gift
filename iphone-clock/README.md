# iPhone Forest Green Clock with Weather

A premium, minimal iPhone clock featuring real-time Korean weather data, fine dust status, and a cinematic landscape layout.

## Features
- **Extreme Scaling**: 75vh+ clock digits for maximum visibility.
- **Real-time Weather**: Accurate data from KMA (Korea Meteorological Administration).
- **Fine Dust Status**: Color-coded air quality information.
- **Cinematic Landscape**: Optimized for iPhone XS/15 Plus landscape mode with a fullscreen second hand.

## How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Start the proxy server: `python proxy_server.py`
3. Open `index.html` in your browser.

## Deployment Note
This project requires a Python backend to handle API requests and bypass CORS. It is recommended to host the entire repository on a service like [Render.com](https://render.com).
